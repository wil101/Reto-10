NTFs ¿Qué son?

NFTS viene del inglés "New Technology File System", que podemos traducir como "Sistema de archivos de nueva tecnología" y se trata del
sistema de archivos que usa la empresa Microsoft en sus sistemas Windows desde hace ya muchos años, desde 2001 exactamente.
